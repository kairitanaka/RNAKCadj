
This script performs the differential expression analysis comparing different tissue types.

The primary goal is to identify a RGP and VGP gene signatures.

This script requires: 

'data/processed/DGE_final_samples.rda'

This scripts creates: 

'tissue_type_degs/results/vobj_fit_contrasts.rda' 
'tissue_type_degs/results/gsea/fgsea_hallmark_results.csv'
'tissue_type_degs/results/gsea/fgsea_reactome_results.csv'
'tissue_type_degs/results/pca_stuff/fit_reduced_list_for_pca.rda'
